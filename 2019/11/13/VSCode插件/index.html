记录以下自己常用的 VSCode 插件

  • Arm Assembly - ARM 汇编语法高亮。
  • File Template - 一个自动生成文件模板的 ode 扩展。
  • Markdown Extended - markdown 扩展。
  • Markdown TOC - 自动生成 toc 。
  • Paste Image - 将图像直接从剪贴板粘贴到 kdown/asciidoc 。
  • Smalise : smali 语法高亮插件。
  • TabNine : AI 代码神器。
  • vscode-icons : - Icons for ual Studio Code。
  • YARA - yara 语法高亮。
  • kite - AI 代码神奇。
  • Doxygen Documentation Generator - ygen 文档注释生成。
  • highlight-words - 选中高亮
