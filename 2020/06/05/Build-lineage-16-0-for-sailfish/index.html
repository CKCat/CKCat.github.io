刷机

重启手机至 bootloader 模式，刷入 recovery 。

然后重启至 recovery 模式。选择 Apply update -> Apply from ADB 再执行下列命令刷机。

如果需要格式化手机内的数据，则需要先选择 Factory Reset -> Format data/factory reset -> Format data ，等待格式化完成，最后再安装上面的方式刷入系统即可。
